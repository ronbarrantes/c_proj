# Fun C Project Ideas

## C + Terminal/TUI

- Number guessing game with `ncurses`
- Todo list TUI
- File browser
- Tiny text editor
- Hex viewer for files
- Terminal dashboard that checks local services
- Roguelike dungeon crawler

## C + Graphics

- SDL2 moving square
- Pong
- Snake
- Breakout
- Paint program with mouse input
- Particle simulator
- Software renderer: draw pixels, lines, triangles yourself
- Wireframe cube
- Tiny Doom-style raycaster

## C + Networking

- TCP echo server
- Tiny HTTP server that serves one HTML file
- Chat server/client
- Basic port scanner using `connect`
- HTTP request inspector
- DNS lookup toy
- Packet sniffer later on Linux

## C + Security Learning

- Build a deliberately vulnerable login checker, then fix it
- Write unsafe string copy examples, then inspect crashes
- Make a tiny password vault using file encryption later
- Write a toy fuzzer that feeds random input to your parser
- Parse a simple file format and harden it

## Suggested First Sequence

1. `ncurses` roguelike: move `@` around a map.
2. SDL2 Pong: window, input, movement, collisions.
3. Tiny HTTP server: sockets, requests, responses.
4. Hex viewer: bytes, files, memory.
5. Software renderer: pixels, lines, triangles.
